# 质量数据地图

**工业制造质量管理入门地图 · Quality Data Map**

---

## 这本书是什么

《质量数据地图》帮你看清：检验与主数据怎么分、SPC 和预警怎么接、追溯怎么闭环，以及 **QMS、MES 质量模块、LIMS** 各管哪一段。

不是标准条文逐条解读，也不是某套软件的操作手册——**能转发给同事的现场科普**，各厂落地以现场质量体系为准。

**主轴：** 采数 → 分析 → 预警/报警 → 闭环（制造是语境，IT 是载体）
