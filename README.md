## 本仓库是什么

这是《质量数据地图》的 **读者向开源镜像**：

- `docs/` — 已发布章节的 Markdown + 配图  
- [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) 构建静态站，由 GitHub Actions 部署 Pages  

**创作与发布 SOP** 在私有书库维护，不在此仓公开。

## 本地预览

```bash
pip install -r requirements.txt
mkdocs serve
```
